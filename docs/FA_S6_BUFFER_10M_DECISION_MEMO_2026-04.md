# S6 緩和：buffer 第二試行（1000万円）決裁メモ

**作成日**: 2026-04-06  
**文書の性質**: **意思決定メモ（コード変更なし）**。観測: `docs/FA_S6_BUFFER_3M_PLAYCHECK_2026-04.md`。第一試行決裁: `docs/FA_S6_BUFFER_INCREASE_DECISION_MEMO_2026-04.md`。設計枠: `docs/OFFSEASON_FA_PAYROLL_BUDGET_BUFFER_PLAN_2026-04.md`。実装正本: `basketball_sim/models/offseason.py` の `_OFFSEASON_FA_PAYROLL_BUDGET_BUFFER`。観測スクリプト: `tools/fa_offer_real_distribution_observer.py`、`tools/fa_offer_diagnostic_observer.py`。

---

## 1. 文書の目的

**300万円 buffer 第一試行**後も、クイック生成・既定条件下で **S6 の `room` 天井＝`final_offer` の全面張り付き**が **300万円にスケールしただけ**であることが観測済み（`FA_S6_BUFFER_3M_PLAYCHECK`）。本メモは **第二試行として buffer を 1000万円に上げるか**を **1行で決裁**し、次コミットの入力にする。

---

## 2. 3,000,000 円第一試行の評価

- **達成**: **30万円張り付きは解消**（同観測系で `<= 300_000` は 0）。決裁どおり **「ゼロ回避から一段上の桁」**には到達。  
- **残課題**: **1920/1920 が `final == 3_000_000`**、高額 FA も同様、D1/D2/D3 に偏りなし＝**構造的に「buffer 天井＝オファー天井」が全域**で、**本格式の交渉芯には届かない**。  
- **総評**: **第一試行として有効だが終着ではない**（playcheck §4〜5 と同旨）。**次ノブを検討する前提は満たしている**。

---

## 3. 1000万円第二試行案

| 観点 | 内容 |
|------|------|
| **効果** | `room_to_budget` の典型天井が **3M → 10M** に上がり、**芯が数百万〜数千万帯の FA**では **3M 張り付きからの脱却**が進む見込み。億超芯は **依然クリップ**されうる。 |
| **実装** | **`offseason.py` の `_OFFSEASON_FA_PAYROLL_BUDGET_BUFFER` を 10,000,000 に変更**するだけでよい可能性が高い（`Offseason.run` の2回呼び出しは据え置き）。`test_offseason_payroll_budget_sync.py` 等の期待値更新が伴う。 |
| **リスク** | **3M より経営ガイドライン・オーナー目標・CPU FA 分布への影響は大きい**。ただし **`max(既存, roster + B)`** の枠は維持され、**キャップ／S1 は別経路**のまま。 |
| **根拠の十分性** | クイック行列で **全面 3M 張り付き**が出ている以上、**「次の一歩として B を上げる」合理性は高い**（理想の最終額の決定ではなく **段階試行**）。 |

**実セーブ／`--seasons 1` 追加観測**: **分布が緩む可能性はある**が、**第二試行を無期限に保留するほどの反証は現時点ではない**。併せて走らせるのは **推奨**だが、**本決裁の前提条件とはしない**（クイック観測＋設計書の段階ノブで足りる）。

---

## 4. 推奨判断

- **1000万円を第二試行の第一候補として採ってよい**。  
- **理由**: (1) **3M は観測上「終着」になっていない**。(2) **変更経路が局所**でロールバックも定数戻しで可能。(3) **BUFFER_PLAN** の **「数百万〜数千万」**の **上側**に位置し、**3M→10M は段階として連続**。(4) **S1 決裁**とは独立。  
- **据え置きのみ**とするのは、**実プレイで 3M で十分と判明した後**が適切だが、**現状の観測メモは逆のシグナル**を出している。

---

## 5. 決裁用の1行結論

**S6 緩和の第二試行として、オフFA直前の `payroll_budget` 同期 buffer は 10,000,000 円（1000万円）に引き上げる。**
