# レギュラーシーズン日程モデル（週次・確定案）

本書は **ユーザー確定の考え方**を記録する。`basketball_sim/models/season.py` の現行実装（ダブルラウンドロビン等）と **必ずしも一致しない**場合がある。スケジュールをコードへ落とすときは本書と `.cursorrules` を正とし、`season.py` を整合させる。

## 方針（要約）

- **レギュラーシーズンは 1 チームあたり 60 試合**（ユーザー確定）。
- **天皇杯の週（R13・R14）はリーグは行わない**（リーグ試合数 0）。杯のみの週として扱う。
- 週次は **R1〜R30** の **30 ラウンド**で、各ラウンドに「月」「主要イベント」「その週のリーグ試合数（1 チームあたり）」を割り当てる。

## リーグ試合数の合計について

下表の「リーグ試合数」を **全ラウンド合計**すると **60**（1 チームあたりレギュラー 60 試合）。R8・R21 は **2**（当初案の 3 から変更し合計を 60 に整合）。
