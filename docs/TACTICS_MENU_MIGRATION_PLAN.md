# 戦術メニュー（旧7ボタン直列）→「プレイスタイル / ローテーション」二枠 — 設計と現状

---

## 現在の実装（本リポジトリの正本）

- **正本（ハブ）**: `basketball_sim/systems/main_menu_view.py` の `open_strategy_window()`。  
- **導線**: 戦術メニューは **「プレイスタイル」** / **「ローテーション」** の **2 つの `LabelFrame`** に整理済み。旧 **7 ボタン1段直列** ではない（ナビ上は **2 枠内**に**複数のオープンボタン**が配列される形）。

| 枠 | 主な入り口（従来ラベル相当） | 主な補足 |
|----|--------------------------------|------------|
| **プレイスタイル** | 基本方針（`Team`）、攻守の傾向（`team_tactics`）、セット傾向（`playbook`） | `PLAYSTYLE_PRESET_DEFS` 由来の**プレイスタイルプリセット**（3 候補）＋`apply_playstyle_preset_with_preset_meta` |
| **ローテーション** | ローテ詳細、起用方針テンプレ、先発・6th・ベンチ、参考役割（個別起用＋`main_role` 等） | `ROTATION_PRESET_DEFS` 由来の**ローテーションプリセット**（3 候補）＋`apply_rotation_preset_with_preset_meta` |

- **`preset_meta` / カスタム表示**は `team_tactics.py` およびハブ上の**プリセット行**に接続（詳細は `TACTICS_PRESET_CUSTOM_STATE_POLICY.md`）。  
- **補助**: `ensure_team_tactics_on_team` 等 — `basketball_sim/systems/team_tactics.py`。

## 設計履歴（導入前のメモ再掲）

- **旧レイアウト**（7 系統のボタンを**1段に並べ**た戦術メニュー）は、**Phase 1 相当の作業**として **2 枠ナビ＋従来 `_open_tactics_*` 再利用** へ**移行済み**（コミット `1957346` 等でハブ表現を更新。詳細は `git` 履歴参照）。  
- 特定日（例: ドキュメント作成日時点）の**スナップショット記述**は、**下記の「当時案」**として**設計比較用**に留める。現行の**ボタン文言・行数**は**コード**を正とする。

**当時案（参考）** — ナビ7本を「プレイスタイル寄せ / ローテーション寄せ」に**見た目だけ**二分し、**`command` を変えない**方針が**推奨**とされていた。→ **後続の 2-LabelFrame 化**はこの方向性と**整合**。
