## 12. 推奨タスクの最小実装案（Cursor 指示骨子）
================================================================================

### 目的

`roster_view.gd` の **`_add_player_row`** で、9列 HBox を **平坦 PanelContainer** でラップし、
**ロスター本格整備の中規模第1手**とする（**細かい margin レーンではない**）。

### 変更してよいファイル

- `godot/scripts/roster_view.gd`（`_add_player_row` のみ）

### 変更してはいけないファイル

- `roster_view.tscn`、Theme `.tres`、`project.godot`
- `_apply_snapshot`、`_add_table_header_row`、`_col_width`
- 他 scene / script / data JSON / export / tests / mock
- README / docs / reports

### 実装方針

1. `_add_player_row` 内で従来どおり **HBox + 9 Label** を組み立てる。
2. **HBox を PanelContainer の子**にし、**Panel に最小 content margin**（例: 4〜6px）または **平坦 bg** を設定。
3. **`_row_list.add_child(panel)`**（従来の `_row_list.add_child(row)` の代わり）。
4. **Phase4SummaryCard の二重ネストは避ける**（TableCard 内の行は **サブカードではなく行帯**）。
5. **表示文言・列幅・tooltip・JSON key は不変**。
